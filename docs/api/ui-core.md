# @atscript/ui-core

Framework-agnostic runtime for form and table definitions from Atscript annotated types.

## Installation

```bash
pnpm add @atscript/ui-core
```

## Overview

`ui-core` reads compiled Atscript type metadata and produces structured form and table definitions. It is the foundation that framework-specific packages (like `vue-form`) build upon.

This package has no dependency on Vue or any other UI framework.
